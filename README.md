# MAX — AI Vetting Workspace

Frontend-first v1 demo for the internal influencer vetting operations platform.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

## Structure

```
src/
├── app/          # routes
├── components/   # shared + ui
├── lib/          # utilities
├── types/        # TypeScript types
└── data/         # mock data
```

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
